Spelling App - About

About

This spelling game was developed initially in Python and then converted to a web project. This version uses the National Curriculum's words list for years 3 and 4.

Further Improvements

  • End of game score review
  • Ability to select different sets of words
  • Ability to change the duration of the game

Credits

The words list is from The National Curriculum.
